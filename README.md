## How to Use

1. **Select a track** from the dropdown menu
2. Click **LOAD** to load it onto the deck
3. Click **â–¶ PLAY** to start playback
4. **Crossfader** - Slide left (Deck A) or right (Deck B)
5. **Pitch** - Speed up/down the track
6. **Volume** - Control loudness
7. **EQ** - Boost/cut bass, mids, or treble
